# Git & GitHub Tutorial Repository

An open repository for learning Git and GitHub.
It contains documents in both Japanese and English, suitable for beginners getting started with Git as well as those looking to integrate with GitHub or explore more advanced workflows.

> 日本語版: [README.md](README.md)

## How to Use This Repository

**Start by reading documents 01–04 in order.** They cover everything from setting up a repository on the server to advanced GitHub integration.
The remaining documents are supplementary — refer to them as needed for command reference or configuration.
